HR Attrition Dashboard

An interactive dashboard built with **Streamlit** to analyze employee attrition trends based on job roles, job levels, tenure, gender, overtime, and more.

This project was created as part of a **Data Analyst course** to practice real-world HR analytics and data visualization.

Features

- Dynamic KPIs (Attrition rate by role, level, tenure)
- Filters by department, overtime, job level
- Visuals: Bar charts, heatmaps, segment breakdowns
- Insights and recommendations based on data

How to Run

```bash
pip install streamlit pandas plotly seaborn matplotlib
streamlit run app.py
